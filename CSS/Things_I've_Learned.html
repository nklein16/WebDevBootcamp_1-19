There are 3 main ways to select elements. There are by element, by id (#), and by class (".") selectors. Of the element types, only block-level elements can be utilized to select portions of HTML for styling by CSS.

IDs

  1. The "id" tag is a way to single out a (single) element for css styling.
  2. A particular id tag can only be used once per page.

    In fact, there are over 30 types of CSS selectors. However, we will focus on five (5) of these additional selector types. They are:

  • Star Selectors (*) - selects everything on the page
  • Descendant Selectors - takes two or more selectors/tag names and chains them together. An example might be to select all anchor tags located inside an li that is located inside a ul. Such a descendant selector would have the following structure: "ul li a { Property: value; }"
  • Adjacent Selectors - only selects selectors/tag names adjacent to other selectors. For instance, we may select only those "ul"s that are on the same level as and immediately follow an h4 like so: "h4 + ul { Property: value; }"
  • Attribute Selectors - A way to select elements based off of any attribute (href, img, input types, etc.). Here's as example: "a[href="www.google.com"]"{ Property: value; }. Another example: "input[type="text"]"{ }.
  • nth of type Selectors - Selects every nth element. Example: Select final "ul" element. Format: ul:nth-of-type(3){ Property: value}

Colors

  • Red
    • Hexadecimal: #FF0000;
    • RGB: rgb(255,0,0);
  • Green
    • Hexadecimal: #00FF00;
    • RGB: rgb(0,255,0);
  • Blue
    • Hexadecimal: #0000FF;
    • RGB: rgb(0,0,255);
  • White
    • Hexadecimal: #FFFFFF;
    • RGB: rgb(255,255,255);
  • Black
    • Hexadecimal: #000000;
    • RGB: rgb(0,0,0);

    Note that there is also something called RGBA that has a fourth channel for the extent to which the letters should be transparent. The A channel ranges from 0.0 - 1.0, so that:
    rgba(100, 0, 100, .1) will be a very transparent purple, whereas rgba(100, 0, 100, 1) will be a completely opaque (with no transparency) purple.*/
